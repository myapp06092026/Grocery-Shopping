Connect Firebase

Paste the firebaseConfig you copied from the Firebase console below, then tap Connect.

Also, in the Firebase console: enable Anonymous sign-in, and set these Firestore rules:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /rooms/{room}/{doc=**} {
      allow read, write: if request.auth != null;
    }
  }
}
